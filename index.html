Devops Glossary
ID# Term Description Reference
1 Devops Devops is the combination of “Dev”(software development) and “ops” (operations). It seeks to integrate the work of both into one cohesive collaboration with the goal of increased integration and shared responsibility Gitlab. What is DevOps? [online]. Available: https://about.gitlab.com/topics/devops/
2 npm “npm” stands for node package manager. Its essentially a free library of commands that can be accessed and used in javascript. Careerfoundry. What is npm? A begginer’s guide [online]. Available: https://careerfoundry.com/en/blog/web-development/what-is-npm/#:~:text=npm%20stands%20for%20Node%20Package,kind%20of%20a%20big%20deal.
3 Repository A repository (in this case, one from github) is a web location where one can store all of a project’s data and files. This data in turn, can be shared with other people who can be given access. The owner is able to manage the repository as seen fit. Github docs. About repositories [online]. Available: https://docs.github.com/en/repositories/creating-and-managing-repositories/about-repositories
4 Cloud A cloud (in the context of computers) are a bunch of servers that can be used over the internet. These servers can be accessed from anyone on earth by the people with the correct permission. This make sit so individuals and companies don’t need to get their own physical servers. Cloudfare. What is cloudcomputing? [online]. Available: https://www.cloudflare.com/learning/cloud/what-is-the-cloud/#:~:text=%22The%20cloud%22%20refers%20to%20servers,centers%20all%20over
5 CLI CLI (command line interface) is the user interface used to actually run programs. It accepts text based commands from the user, and the computer runs those commands. The commands themselves follow a set of predetermined commands. Techtarget. Command-line interface (CLI) [online]. 6 Microservices Microservices is a style of online architecture where the application is made as a collection of smaller services. each of these services can be independently used and is only loosely coupled. Microservice architecture. What are microservices? [online]. Available: https://microservices.io/
7 SDLC SDLC (software development life-cycles) is the traditional life cycle of development for software. Available: https://www.synopsys.com/glossary/what-is-sdlc.html#:~:text=Definition,the%20shortest%20possible%20production%20time.
